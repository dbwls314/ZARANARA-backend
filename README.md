# 27-1st-ZARANARA-backend
> 가구 및 생활 소품 판매를 하는 다국적 기업인 [**ZARA HOME**](https://www.zarahome.com/kr/)를 클론 코딩한 **ZARANARA**팀 입니다!

## 구현 기능
- 회원가입 [ 담당 기능 ]
- 로그인 [ 담당 기능 ]
- 제품 리스트
- 제품 상세 페이지
- 장바구니
- 유저 별로 '좋아요(Like)' 토글 기능 구현
- CSV 데이터 작성 및 데이터 입력

<br>

## User [ 담당 구현 기능 ]
- 이메일 유효성 검사 및 비밀번호 조건부여 회원가입
- bcrypt 비밀번호 암호화
- 로그인시 JWT 토큰 발행 및 토큰 검사
